# SYSU-SportsAssociationManagementCenter
That's a respository dedicated to storing information about East Campus Sports Association Management Center from SYSU.

### 社团介绍：
东校园体育社团管理协会，这是一个致力于服务中大体育事业的学生组织

### 存储内容：
- 社团工作文档
- 会议记录
- 活动记录
